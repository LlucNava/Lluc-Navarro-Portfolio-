Sorting Visualizer

This project was created at Karelia University (Joensuu, Finland) to practice Java and CSS. It visualizes sorting algorithms like Bubble Sort and Insertion Sort. Bars are randomized with each shuffle, and the sorting process is shown through animations, helping to understand how different algorithms sort arrays step by step.

Visual Novel

This is a horror interactive novel developed as part of a team project. Together, we planned the concept, created the videos, and prepared all the assets. I was responsible for programming the experience using JavaScript and the p5.js library.
